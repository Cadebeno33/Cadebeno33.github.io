Spaghetti Plot of a Hurricane approximately 5 days from landfall

Ice thickness prediction on a Lake

Within this phot it shows the growth of ice on Lake Mendota. This graph is taking in a variety of factors all ran in Matlab. As you can see, the ice thickness is all based on when the lake is completely covered in a layer of ice. Within my LaTeX document, I talk aobut the applications that this model has. Essentially, once a lake freezes over completely, which happens in 7-10 days of below freezing temperatures, we can accurately predict the ice thickness of a lake. It does this by taking into account the meteorlogical data for the future, the snow fall, and working on small to medium sized lakes. Using this, you can accurately predict the ice thickness multiple days in advance. This model has numerous implications and if you are interested, please reach out and I can send you my write up.

Matlab code to accurately predict ice thickness on a lake in advance

In this code, we can incorporate when snow fall occurs, when the air temperature matches the water, and update the ice thickness over each day. By setting some paramters, establishing a goal, and using this code I was able to make an accurate model based on the weather futuristic predictions. Quite easily, we could autimatically reference an external weather app to give us the most accurate predictions for our ice. This code has applications in studying lakes, recreation, and safety.
